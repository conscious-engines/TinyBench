# MLCChat — Android Benchmarking App

See the [root README](../../README.md) for full build and run instructions.

**Quick reference — open this directory as a project in Android Studio:**

```
android/MLCChat/
```

Prerequisites before opening in Android Studio:
1. Model compiled and assets placed in `app/src/main/assets/` (Steps 3–4 in root README)
2. Native JNI libraries built and placed in `mlc4j/output/jniLibs/` (Step 5 in root README)
3. `mlc4j/libs/tvm4j-core-0.0.1-SNAPSHOT.jar` present (Step 5 in root README)
